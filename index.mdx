Arcoms 1 is a production-ready landing page template built on Next.js 16, React 19, and Tailwind CSS v4. It delivers an immersive visual experience through Spline 3D scenes, a WebGL globe, fluid Motion animations, and a live multi-theme system — all optimized for performance.

## What's included

<CardGroup cols={2}>
  <Card title="Multi-theme system" icon="moon">
    Three visual modes (Light, Dark, Colorblind) and six accent colors switchable at runtime without a page reload.
  </Card>
  <Card title="3D Spline scenes" icon="cube">
    Interactive 3D models embedded via Spline, rendered client-side with no SSR jank.
  </Card>
  <Card title="WebGL globe" icon="globe">
    A COBE-powered WebGL globe and a D3.js + TopoJSON SVG wireframe globe for the contact page.
  </Card>
  <Card title="GPU-optimized animations" icon="bolt">
    All transitions use only `opacity` and `transform` — composited properties that never trigger layout or paint.
  </Card>
  <Card title="Bento grid" icon="grid">
    Responsive bento-layout feature section with floating image cards and animated stat widgets.
  </Card>
  <Card title="Interactive pricing" icon="tag">
    A four-tier pricing table with monthly/yearly toggle and click-to-select plan highlighting.
  </Card>
</CardGroup>

## Getting started

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/EmirPolito/Arcoms-1.git
    cd Arcoms-1
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Start the dev server">
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser.
  </Step>
  <Step title="Customize your theme">
    Edit `app/globals.css` to update CSS tokens, or use the in-browser theme switcher to preview all modes live.
  </Step>
</Steps>
