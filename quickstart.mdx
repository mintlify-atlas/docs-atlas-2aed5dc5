title: "Quick start"
description: "Clone the repository, install dependencies, and have a local dev server running in under two minutes."
---

<Steps>
  <Step title="Clone the repository">
    Clone the Arcoms 1 repository from GitHub and move into the project directory.

    ```bash
    git clone https://github.com/EmirPolito/Arcoms-1.git
    cd Arcoms-1
    ```
  </Step>

  <Step title="Install dependencies">
    Install all Node.js dependencies with npm. The project has no optional peer-dependency prompts or postinstall scripts to configure.

    ```bash
    npm install
    ```

    This installs all `dependencies` and `devDependencies` declared in `package.json`, including Tailwind CSS v4, Motion, Spline, COBE, and all Radix UI packages.
  </Step>

  <Step title="Start the dev server">
    Start the Next.js development server. Turbopack is enabled by default in Next.js 16 for fast HMR.

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) in your browser. The page hot-reloads on every file save — including changes to `globals.css` theme tokens.

    <Note>
      The first cold start may take a few seconds as Turbopack compiles the initial bundle. Subsequent reloads are near-instant.
    </Note>
  </Step>

  <Step title="Build for production">
    When you are ready to deploy, build an optimised production bundle and verify it starts correctly.

    ```bash
    npm run build
    npm start
    ```

    `npm run build` runs `next build`, which produces a `.next/` output directory with server components, static assets, and image optimisation manifests. `npm start` serves that build locally on port 3000.

    <Tip>
      Run `npm run lint` before deploying to catch any ESLint issues. The project uses `eslint-config-next` with Core Web Vitals and TypeScript rules enforced.
    </Tip>
  </Step>
</Steps>

## System requirements

| Requirement | Minimum version |
|---|---|
| **Node.js** | 18.18.0 (required by Next.js 16) |
| **npm** | 9.x or later |
| **Git** | Any recent version |

<Warning>
  Next.js 16 requires **Node.js 18.18.0 or later**. Running `node -v` before cloning is recommended. Older Node versions will fail at `npm run dev` with a version compatibility error.
</Warning>

## Available scripts

All scripts are defined in `package.json`:

```json
{
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "eslint"
  }
}
```

| Script | Command | Description |
|---|---|---|
| `dev` | `npm run dev` | Start the Turbopack development server |
| `build` | `npm run build` | Produce an optimised production build |
| `start` | `npm start` | Serve the production build locally |
| `lint` | `npm run lint` | Run ESLint with Next.js + TypeScript rules |
