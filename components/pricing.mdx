`DemoInteractiva` renders a full pricing section. It manages two pieces of local state — `isYearly` (billing period toggle) and `selectedPlan` (which card is highlighted) — and derives all displayed prices from an inline `plans` array. Cards animate in on scroll using Framer Motion's `whileInView`.

```tsx
import DemoInteractiva from "@/components/pricing";

<DemoInteractiva />
```

No props — all plans, prices, and feature lists live in the `plans` array inside
`components/pricing.tsx`.

## Interactive features

### Monthly / yearly toggle

Two `<button>` elements inside a `rounded-full bg-muted` pill switch `isYearly`
between `false` (monthly) and `true` (yearly). The active button receives
`bg-price-btn-bg text-price-btn-txt shadow-lg`; the inactive button uses
`text-price-desc hover:text-price-ttl`.

The displayed price switches between `plan.priceMonthly` and `plan.priceYearly`,
and the suffix switches between `/mes` and `/año`. The yearly option shows a
`(-20%)` hint.

```tsx
<button onClick={() => setIsYearly(false)}>Mensual</button>
<button onClick={() => setIsYearly(true)}>
  Anual <span className="text-xs opacity-75">(-20%)</span>
</button>
```

### Click-to-select plan

Each card has an `onClick` handler that calls `setSelectedPlan(index)`. The
selected card receives:

```
border-price-btn-bg bg-price-btn-bg/5 shadow-lg scale-105 z-10
```

Unselected cards use `border-price-border bg-price-card`. The Pro card
(`index === 1`) is pre-selected on mount (`useState(1)`).

### Scroll entrance animation

Each card is a `motion.div` that animates from `{ opacity: 0, y: 30 }` to
`{ opacity: 1, y: 0 }` when it enters the viewport. Cards stagger with a
`delay: index * 0.1` (0 ms, 100 ms, 200 ms, 300 ms):

```tsx
<motion.div
  initial={{ opacity: 0, y: 30 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true }}
  transition={{ duration: 0.5, delay: index * 0.1 }}
>
```

## Customization

### Add or remove a plan

Edit the `plans` array in `components/pricing.tsx`. Each entry follows this shape:

```typescript
{
  name: string;          // Display name
  description: string;  // Subtitle below the name
  priceMonthly: number; // Monthly price in USD
  priceYearly: number;  // Yearly price in USD
  features: string[];   // Feature bullet list
  cta: string;          // Button label
  popular: boolean;     // Show the "Popular" badge
}
```

### Change prices

Update `priceMonthly` and `priceYearly` on the relevant plan object. The toggle
and display logic derives everything from these two fields automatically.

### Change the default selected plan

The second argument to `useState` controls which plan is highlighted on first render:

```tsx
// Highlights "Pro" (index 1) by default
const [selectedPlan, setSelectedPlan] = useState(1);
```

Change it to `0` for Gratis, `2` for Equipo, or `3` for Empresa.

### CSS tokens used

| Token | Purpose |
|---|---|
| `bg-price-bg` | Page/section background |
| `text-price-ttl` | Primary text color |
| `text-price-desc` | Muted / secondary text |
| `bg-price-card` | Unselected card background |
| `border-price-border` | Unselected card border |
| `bg-price-btn-bg` | Primary button & selected border |
| `text-price-btn-txt` | Text on primary button |

Define these in your Tailwind v4 theme.
